AsyncHttp¶

Handle asynchronous http requests

Usage:

Request to PS2 api: json_result_file = await api_request_and_retry(url) Standard HTTP request: result_code = await request_code(url)

exception modules.asynchttp.ApiNotReachable(url)¶

Bases: Exception

Custom API request exception.

Attributes:

url: Url of the failed request.

async modules.asynchttp.api_request_and_retry(url: str, retries: int = 3) → dict¶

Try to query Planetside2 API.

Parameters

  • retries – (Optional, default: 3) Number of retries.

  • url – URL to get.

Returns

Json dictionary returned by the API.

Raise

ApiNotReachable if the request failed.

async modules.asynchttp.request_code(url: str) → int¶

Get the url requested.

Parameters

url – URL to get.

Returns

HTTP code returned.
